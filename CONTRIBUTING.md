# Участие в проекте

Спасибо за интерес к проекту.

- Создавайте отдельную ветку для каждого изменения.
- Не добавляйте в репозиторий пароли, токены, IP-адреса, домены и ссылки подписок.
- Для изменений скриптов запускайте `bash -n install-xhttp-vps.sh`, `bash -n finish-xhttp-vps.sh` и ShellCheck. Эти проверки также выполняются автоматически в Pull Request.
- Описывайте в PR, что изменено и как это проверялось.
- Изменения, которые могут удалить данные, ослабить защиту или изменить сетевую схему, требуют отдельного обоснования.

Сообщения об уязвимостях — по правилам из [SECURITY.md](SECURITY.md).
